解决windows窗口拖出屏幕拖不回来的问题

文章发布时间:
2023-04-02

最后更新时间:
2023-04-02

windows 窗口拖动到电脑边缘后拖不回来，可以用键盘快捷键救急

方法

  1. alt + tab 切换到需要拖动的窗口

  2. alt + 空格 打开窗口控制选项

  3. m 选择移动窗口，或 r 退出全屏

  4. 把窗口拖回来
