一、爬取数据

爬数据就是用 post 和 get 来获取数据，主要包括打印页面和用 xpath 定位数据两方面操作。

1. 打印页面

调用 get/post 返回值 response 中的 text 方法就能显示页面数据

也可以使用 post 方法，附带 data 和 header

乱码问题

有时候 response.text 是乱码，很可能是编码问题导致的，修改 response 的编码格式然后再打开即可
