在Linux跑Python脚本的指令，以及相关的控制命令

脚本运行指令

命令如下：

  • nohup：程序不被挂起

  • -u：不启用缓存，实时打印信息到输出文件

    不加-u会导致日志文件不会实时刷新

  • >：重定向打印信息

  • 2>&1：将 stderr 转化为 stdout (0=stdin, 1=stdout, 2=stderr)
