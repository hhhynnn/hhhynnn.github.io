注：username 是 github 账户名，如我的就是 hhhynnn.github.io

1.3. 配置 hexo

1.3.1. 在本地搭建博客

step1. 任意位置执行npm install hexo-cli -g 下载 hexo 组件

step2. 在想建立博客的目录下执行 hexo init ZenithBlog 初始化一个博客文件夹ZenithBlog

step3. 在ZenithBlog/目录下执行 npm install 下载所需组件

step4. 执行 hexo server -p 10809 即可在 localhost:10809 浏览自己的博客

1.3.2. 将博客部署到 github

step1. 在ZenithBlog/目录下执行 找不到npm install hexo-deployer-git --save 下载必要组件

step2. 修改在ZenithBlog/_config.yml 中的 deploy值

3.2. 更换主题方法

假设想换的主题叫 “Kartos-Rebirth”

地址： Kratos-Rebirth 文档

step1. 下载主题文件到./themes/Kartos-Rebirth

step2. 修改./_config.yml 文件中的 theme值

step3. 根据主题作者的文档进行相关配置

e.g. Kartos-Rebirth 主题作者要求修改Kartos-Rebirth/_config.yml.example 为 Kartos-Rebirth/_config.yml

step4. 在./目录下依次执行 hexo clean , hexo g , hexo d
