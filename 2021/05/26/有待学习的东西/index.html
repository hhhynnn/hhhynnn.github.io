Java修炼之路

2021-05-26

笔记最终的目的是为了提高效率，不是为了复制整理，形式不是一切

一级标题：核心内容，二级标题&三级标题：主题，四级标题：常见应用和细节

1. 学习重点

1.1. 基础课程

四大基础分别是：计算机网络、计算机组成原理、操作系统、数据结构。

计算机网络

玩IT不懂网络可以直接白白了，必须得认真学，参考书目如下：

《计算机网络》+《图解 HTTP》

计算机组成原理

计组老师讲得太水，但这门课非常重要，既能帮助对代码进行优化，又能加快学习编程的速度。

理解底层对理解上层有着极强的促进作用，需要认真学习，参考书目如下：

《计算机组成原理》

操作系统

(暂时还没研究

数据结构

数据结构的重要性不言而喻，算法的根基！

《数据结构与算法分析 : Java 语言描述》

1.2 算法

算法就是平时多刷多练，参考资料如下：

《剑指 offer》+ LeetCode

1.3 Java 核心技术

Java 基础（语法+JVM+多线程）

多门语言不如单门深入，python很火但是不适合写大项目，并且跑得太慢，抓住Java往深了学比较好。

参考书目如下：

《深入理解 JVM 虚拟机》+《Java8 实战》+《Effective Java》

Spring 框架

spring即将被淘汰，但仍然在使用，参考书目：

《Spring+MyBatis 企业应用实战》+《分布式消息中间件实践》

1.4 其它基础

数据库

数据库不止是sql系列，还有redis需要了解，具体参考：《Redis 入门》

代码架构

写项目的关键，代码架构和算法并驾齐驱，在工作中甚至比算法重要，需要给予足够的重视

参考书目：

《重构》+《设计模式》+《分布式消息中间件实践》
